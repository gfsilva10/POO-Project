# SpotifUM (POO)

## Grade: 18/20 
---

## Project Overview
This system was developed as part of the **Object Oriented Programming (OOP)** course. It provides a comprehensive solution for managing a fast-food establishment, balancing the needs of staff management with customer order processing.

---

## 🎯 Learning Outcomes
* **Design Excellence:** Focus on software patterns to ensure code maintainability and scalability.
* **Proficiency in Object-Oriented Programming:** Application of OOP concepts to build a robust and extensible system.

---

## 🚀 Key Features
* **Order Management:** Complete flow for placing, tracking, and completing customer orders.
* **Inventory & Menu Control:** Tools for managers to update products, prices, and stock levels.
* **Staff Coordination:** Modules for employee management and role-based access control.
* **Reliable Operations:** Built-in validation to prevent order errors and ensure data consistency.

---

## 🏗️ Architecture & Implementation
* **Language:** Java.
* **Patterns:** **Facade Pattern** for business logic organization.
* **Diagrams:** Logical and physical system organization, with a focus on object-oriented implementation.

---

## 🛠️ How to Run
1. **Clone** the repository: `git clone https://github.com/gfsilva10/POO-Project`.
2. **Setup:** Ensure that the Java Development Kit (JDK) is installed.
3. **Compile:** Build the source files located in the `src` directory.
4. **Launch:** Run the main application class to access the menu.
